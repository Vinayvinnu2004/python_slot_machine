# python_slot_machine
This project is a console based Slot Machine game built using Python. The player starts with a balance, places a bet, and spins three reels that animate continuously for a fixed duration. Wins and losses are decided by matching symbols, with sound effects and balance updates providing a realistic terminal based gaming experience.

FEATURES:
1.Real-time sound effects using winsound
2.Betting system with balance validation
3.Jackpot and win rewards
4.Loss handling and game-over condition
5.Play again or exit option

TECHNOLOGIES USED:
1.Python 3
2.random module (symbol selection)
3.time module (spin duration)
